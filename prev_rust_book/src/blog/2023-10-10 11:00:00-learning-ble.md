# Learning BLE

I started learn about BLE with the book "Intro to Bluetooth Low Energy: The easiest way to learn BLE" by Mohammad Afaneh.
It's a well-written book with clear explanations, and it's enought to get started with BLE applications on microcontrollers.


There are a lot of Bluetooth versions nowadays, but I am specifically interested in finding directions. So, I tried to create an antenna array, which worked, but it has some issues.

My next step is to delve into BLE. In my opinion, it involves undertanding the BLE stack. [Someone](https://stackoverflow.com/questions/42476466/open-source-implementations-of-ble-stack) has already inquired about an open-soure stack.

And as a result trying to start playing with Apache [NimBLE](https://mynewt.apache.org/latest/network).
