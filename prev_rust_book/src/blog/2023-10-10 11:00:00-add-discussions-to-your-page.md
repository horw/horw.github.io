# Add discussion on your blog/page

While browsing other users' GitHub pages, I found an interesting thing: there is a comment section! And it was labeled 'powered by giscus.' It got me interested. Giscus is a plugin for working with GitHub repository discussion section.

You can set up the Discus using this [tutorial](https://www.freecodecamp.org/news/github-discussions-as-chat-system/).
